# kate.amrik.co.uk

This is the source code for our wedding website, [kate.amrik.co.uk](https://kate.amrik.co.uk).

The website is built using [Hugo](https://gohugo.io/), a static site generator written in Go.

Feel free to use this code as a template for your own wedding website. If you do, please let us know! We'd love to see what you come up with.
